## Installation

### Environment Variables

| Name                    | Required? | Type    | Default |
|-------------------------|-----------|---------|---------|
| SALTMARSH_DISCORD_TOKEN | Yes       | String  |         |
| DEVELOPER_MODE          | No        | Boolean | True    |


### Docker-Compose
You can simply run this bot using `docker compose up`.
